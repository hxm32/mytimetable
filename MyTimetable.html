Two ways to log in. The bookmarklet is easiest — set it up once and use it every time.

Option 1 — Bookmarklet (recommended, one-time setup)
Drag the button below to your bookmarks bar. Then whenever your session expires, just go to CaulfieldLife and click it.

⚑ Connect to Timetable
  1. Drag the button above to your browser's bookmarks bar. If you can't find it, press Command + Shift + B (Mac) or Ctrl + Shift + B (Windows) to show it in Chrome.
  2. Go to caulfieldlife.com.au and log in
  3. Click the Connect to Timetable bookmark — a message will appear saying the interceptor is active
  4. Click on My Timetable in CaulfieldLife to trigger a network request — you'll be redirected here automatically
or

Go to caulfieldlife.com.au, log in, open DevTools → Network → filter ClassData → reload → click the request → Request Headers → copy x-community-token.
