# mytimetable
A unofficial version of CaulfieldLife. This version loads faster and has customisable themes, as well as a search feature for searching classes, locations and teachers.
**To use, download the file `MyTimetable.html` and open it with any browser, or visit the site https://mytimetable.free.nf/**

Again, this is a unofficial project. If something goes wrong, please contact me instead of the school. My disocrd username is `hxm32`. Read the instructions on the code/website before contacting me, it usually explains how to fix most of the problems.
