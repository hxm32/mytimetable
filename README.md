# mytimetable

An unofficial, faster alternative to CaulfieldLife with customisable themes, 
class search, and more.

### Features
- 8 built-in themes + fully customisable colours
- Search classes by subject, teacher, or room across a 5-week window
- Active class highlight with live progress bar
- Next class button to easily know what you have next
- Bookmarklet login — no DevTools required

### To use, visit https://mytimetable.free.nf

Alternatively, download `MyTimetable.html` and open it in any browser. 
This method usually doesn't support the bookmarklet login.

I've also documented what I've discovered about the CaulfieldLife API 
in `API documentation.md`.
