# mytimetable

Disclaimer: This project is not affiliated with, endorsed by, or connected to Caulfield Grammar School or the CaulfieldLife platform in any way. It is an independent, unofficial tool built for personal use. Use it at your own risk. If something breaks, contact me — not the school. My discord username is `hxm32`. This tool only accesses data that the authenticated user is already authorised to view. No school infrastructure is used to run this project. All processing happens locally in the user's browser.

### To use, visit the site https://mytimetable.free.nf/

Alternatively, you can download `MyTimetable.html` and open it with any browser. This method usually doesn't support the bookmarklet login.

I've also documented what i've discovered about the API requests that CaulfieldLife uses in `API documentation.md`.
